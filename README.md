Getting Started
---------------

Open `index.html` directly in your browser to preview.

Structure
---------

- `index.html`: Main page with sections: Hero, About, Skills, Projects, Experience, Contact
- `assets/css/styles.css`: Theme, layout, responsive styles (light/dark supported)
- `assets/js/script.js`: Navigation, smooth scroll, theme toggle
- `assets/img/`: Place images and favicon here (use `favicon.png`)
- `assets/docs/`: Place your resume as `Udit-Kumar-Singh-Resume.pdf`

Customize Content
-----------------

1) Basic info
- In `index.html`, update:
  - Brand name in header
  - Hero title/subtitle and badges
  - About paragraph and Quick Facts
  - Skills chips
  - Project cards (title, description, meta, links)
  - Experience timeline
  - Social links in footer
  - Contact form email (optional)

2) Resume
- Export your resume PDF to `assets/docs/Udit-Kumar-Singh-Resume.pdf`
- Update the filename in `index.html` if you prefer a different name

3) Favicon
- Add `assets/img/favicon.png` (at least 64x64)

4) Colors and theme
- Adjust CSS variables in `assets/css/styles.css` under `:root` and `[data-theme="light"]`

Netlify Form (optional)
-----------------------
The contact form includes `data-netlify="true"` for easy static hosting form handling.
- If not using Netlify, consider replacing the form with a mailto link or backend endpoint.

Deployment
----------
- GitHub Pages: Push to a repo and enable Pages (root as source)
- Netlify/Vercel: Drag-and-drop the folder or connect the repo (no build step required)

Accessibility
-------------
- Includes skip link, semantic tags, focus styles, and color contrast aware palette.
